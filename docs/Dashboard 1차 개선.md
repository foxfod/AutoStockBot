# 개요
- 현재 구현되어 있는 Dashboard를 개선한다. 
## 기존 Dashboard 구분
- 기존 Dashboard를 개선하기 위해 각 섹션에 명칭을 부여 하여 개선명령을 하기 용이하게 한다. 

### Header 
- 맨 위에 로고와 대시보드 제목이 표시되되고 연결 상태, 시간을 표시하는 부분

### Control Panel
- 일시정지, 재시작, 종료 버튼

### Stock Info Panel
- Current mode / Active trades / Kr Budget / Us Budget

### Live Console Output
- 실시간 로그를 확인 할 수 있는 부분

### Active Positions
- 보유하고 있는 정복에 대한 정보를 표시하는 부분.


# 개선 사항.
- 아래 부터 개선 혹은 추가 할 사항을 기술한다.

# Market Info (신규 추가)
- Header와 Stock Info panel 사이에 신규 추가
- 다양한 시장에 대한 정보를 표시한다.
- 해당 정보는 실시간으로 가져오기 어려운 경우, 적당한 시간 간격을 두고 가져올 수 있도록 한다. 
- 단, 가져온 시간을 패널의 한쪽 구석에 표시한다. 

## 표시정보
- 시장정보 (현재 수치)
    - 한국장 (코스피, 코스닥)
    - 미국장 ( NASDAQ, 다우, SP500)    
    * 수치와 금일 변동량을 표시한다. 
    * 상기 수치는 한투 연동하여 표시하고, 만약 한투에서 데이터를 가져오지 못하거나 제공하지 않는 경우, 다른 API를 통해 가져와서 표시한다.
        단, 실시간 정보가 아닌 경우, 해당 정보가 언제 정보인지를 정확하게 표시해야 한다.
    - 시장이 열리지 않은 경우, 전일 종가를 표시한다.
    
    * 한국장과 미국장의 각 시장의 카운트 시간을 표시한다. 
        마감된 경우, 다음 오픈 까지 남은 시간을 표시한다.
        현재 장 중에는 마감까지의 남은 시간을 표시한다. 

- 환율 정보
    - 달러/원 환율
    - 유로/원 환율
    - 엔/원 환율
    * 상기 수치는 한투 연동하여 표시하고, 만약 한투에서 데이터를 가져오지 못하거나 제공하지 않는 경우, 다른 API를 통해 가져와서 표시한다.
        단, 실시간 정보가 아닌 경우, 해당 정보가 언제 정보인지를 정확하게 표시해야 한다.

- 원자재 정보
    - 금 시세
    - 은 시세
    - 원유 시세
    * 상기 수치는 한투 연동하여 표시하고, 만약 한투에서 데이터를 가져오지 못하거나 제공하지 않는 경우, 다른 API를 통해 가져와서 표시한다.
        단, 실시간 정보가 아닌 경우, 해당 정보가 언제 정보인지를 정확하게 표시해야 한다.


# 보유 종목
    - 보유종목 클릭 시 해당 보유 종복의 상세 정보를 확인 할 수 있도록 한다. 
    
## 보유 종목 평가금
- 현재 보유 종목수
- 현재 보유종목의 총 평가 금액
- 현재 보유종목의 총 수익률
- 현재 보유종목의 총 평가 손익



## 슬롯 정보
    - 슬롯 수를 표시한다. 
    - 슬롯은 UI 형태로 표시한다.
    - 비어 있는 경우에도 빈 슬롯을 표시한다. 
    - 수익이 나는 슬롯은 붉은색으로 슬롯 표시.
    - 손해가 나는 슬롯은 파란색으로 슬롯 표시.
    (슬롯은 한국장과 미국장 별도로 사용된다.)
    
    ### 슬롯관리 (기능 추가)
        - +,- 버튼을 클릭 하여 빈 슬롯을 줄이거나 늘릴 수 있다. 
        - 자동으로 설정된 슬롯의 수 변경이 가능 하도록 한다. 
        - 변경된 슬롯수는 해당 당일에만 적용된다. 
    
    ### 슬롯에 표시되는 정보
        - 종목명
        - 티커        
        - 시장
        - 구매 평단가            
        - 보유 수량
        - 현재가
        - 수익률
            아래 수익금액 표시
        - 미국장이라면, 모든 금액 아래 원화로 계산하여 원화표시)
    

## 팝업 내용
    ###  보유 종목의 기본 정보 
        (종목명, 티커, 시장)
    ### 보유 종목의 상세 정보
        - 보유 종목의 평단가, 현재가, 수익률, 수익금액, 보유수량, 총 매수금액, 총 평가금액, 총 평가손익, 총 평가수익률
        - 목표 판매가, 손절가        
    ### 보유 종목의 거래 히스토리
        - 매수일시, 매수시 금액, 매수수량, 매수 평가 금액
    
    ### 기능
        - 종목 분석 
            (클릭 시, 해당 종목에 대한 ai 분석자료를 확인 할수 있다.)
            분석 자료에는 예상 목표가격도 함께 표시된다.
        - 종목 거래
            즉시 판매                
                - 해당 종목을 시장가로 즉시 판매 하는 기능이다. 
                - 클릭 시, 판매 수량을 확인 하는 팝업이 표시된다. (숫자 입력란과 모두 판매 버튼이 제공된다. )
                - 판매 클릭 시, 정말 모두 판매 하는지 확인 하는 팝업이 표시된다. 
                - 확인 클릭 시 해당 종목을 시장가로 즉시 판매 한다. 
            목표가 수정
                - 해당 종목의 목표가를 수정 하는 기능이다. 
                - 클릭 시, 목표가를 확인 하는 팝업이 표시된다. 
                - 수정 클릭 시, 정말 목표가를 수정 하는지 확인 하는 팝업이 표시된다. 
                - 확인 클릭 시 해당 종목의 목표가를 수정 한다. 
            손절가 수정
                - 해당 종목의 손절가를 수정 하는 기능이다. 
                - 클릭 시, 손절가를 확인 하는 팝업이 표시된다. 
                - 수정 클릭 시, 정말 손절가를 수정 하는지 확인 하는 팝업이 표시된다. 
                - 확인 클릭 시 해당 종목의 손절가를 수정 한다.
